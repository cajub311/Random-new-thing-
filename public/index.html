      OpenClaw

      Free, local-first, open-source AI assistant. Runs on Ollama & LM Studio, falls back to keyless cloud providers.

      🎉 No API key required. If you have Ollama or LM Studio running locally, OpenClaw will auto-connect and keep everything private on your machine. Otherwise the free keyless Pollinations provider kicks in. Switch to Agent mode for web search, files, images, memory, and more.

      Free AI APIs may have rate limits. CloudClaw never stores your messages or API keys.
