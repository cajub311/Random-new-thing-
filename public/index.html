    OpenClaw

    Free, open-source AI assistant. Keyless Pollinations by default; add keys (e.g. Groq) in the sidebar for faster models.

    🎉 No API key required to start — the free keyless Pollinations provider works immediately. Add a Groq API key in the sidebar for faster responses; your key stays in this browser. Switch to Agent mode for web search, files, images, memory, and more.

    Free AI APIs may have rate limits. OpenClaw stores chats locally in your browser and never transmits your keys anywhere except the chosen provider.
